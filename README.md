## Sobre mí
Analista de datos con un perfil híbrido y versátil que combina una sólida base en administración, contabilidad y gestión operativa con competencias técnicas en análisis de datos. Experiencia en atención al cliente, control de inventarios y análisis de KPIs comerciales y financieros. Enfoque práctico orientado a la organización de la información y la creación de informes visuales e insights accionables que impulsan la eficiencia operativa y facilitan la toma de decisiones..

- Gestión Operativa y Digital: Capacidad para administrar flujos de trabajo en entornos digitales, garantizando el registro correcto, la limpieza de datos y el soporte técnico necesario para el funcionamiento diario del     departamento.
- Automatización con Python (OOP): Desarrollo de scripts bajo estándares de Programación Orientada a Objetos para automatizar tareas repetitivas, reduciendo el error manual y los tiempos de ejecución.
- Integridad de Datos (SQL & Git): Diseño y consulta de bases de datos mediante SQL, asegurando la normalización de la información. Uso profesional de Git para el control de versiones y la trazabilidad de cada desarrollo.
